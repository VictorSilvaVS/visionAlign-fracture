# Frequently Asked Questions

Simple answers to the most common questions about how the system works.

---

### How do the two systems work together?

Imagine an inspection team with two people:

*   **VisionAlign:** Quickly locates the can on the conveyor belt, ignoring vibrations or misalignments.
*   **VisionFracture:** Receives the area isolated by the first system and focuses exclusively on finding micro-cracks and reading the machine code.

---

### Can the system read codes on dirty cans?

Yes. Artificial intelligence solves this challenge in two steps:

*   **Precision Focus:** The system isolates only the code area, creating a clean image for analysis.
*   **Structural Vision:** The intelligence ignores spots and oil, focusing only on the actual shape of the characters engraved in the metal.

---

### How does the system get smarter over time?

The system works like an apprentice that specializes more each day:

*   **Uncertainty Capture:** Every time the AI encounters something new or doubtful, it automatically saves that image.
*   **Self-Adjustment:** These new images are used to re-train the system, teaching it about new variations of defects.
*   **Constant Evolution:** The intelligence learns without the need for a programmer to change the code.

---

### Is the system secure?

Yes. Security is handled with industrial standards:

*   **Local Processing:** All processing is done inside the factory. Photos do not go to the internet.
*   **Privacy:** There is no mandatory cloud connection, ensuring total control of your data.
*   **Encryption:** All internal communication is protected by digital keys.

---

### What is the Global Brain?

The Global Brain is the technology that allows all machines on the line to share the same intelligence:

*   **Shared Intelligence:** If one machine learns to identify a new type of flaw, all other machines receive this update automatically.
*   **Total Standardization:** It ensures that all inspection points in the factory follow the same criteria and quality parameters.
*   **Evolution via OTX:** It uses Intel training technology so that knowledge generated on one line benefits the entire factory.

---

### Do I need to pay a monthly fee for using AI?

No. The system was built on open technologies:

*   **No License Costs:** We use Intel tools that are free for industrial use.
*   **No Monthly Fees:** You do not pay recurring fees for using artificial intelligence.
*   **One-Time Investment:** The system is yours and operates independently.
